サーバ側とクライアント側
クライアント側（MySQL環境内）：select @@local_infile;
set persist local_infile=1;（（ローカル韻ファイル許可フラグ）
select @@local_infile;

↓だったかも
mysql --local-infile=1 -u root -p1

サーバー側（コマンドプロンプト）：
コマンドプロンプトでmysql使えるようにするためにパス通す必要があった
mysql -u root -p --enable-local-infile
たぶんこんなんやった

んでMySQL実行したら文字コードエラーやらいろいろでた。
ファイルのパスは\じゃなくて/でやるべきらしい
